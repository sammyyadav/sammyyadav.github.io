    PROJECTS

    PROJECT

    Obstacle Avoidance Robot

    This our forst project as an engineering student in first semester for our Engineering Clinincs. With Rakesh D, Logesh and Densiton as Team members we carried out this project by using Arduino Uno , servo motor and ultrasonic sensor on a two wheeled motor. We displayed the project in front of all first year students as we volunteered to help them in their project work during the hackathon conducted in the next day of our project display.

    PROJECT

    Quadruped Robot

    As its name suggests, Quadruped Robots have four legs or limbs and follow the gait patterns of quadruped animals. They are faster and more stable than biped robots. The body was 3D printed and 9 servo motors are used for the motion with Arduino Uno as controller

    PROJECT

    Autonomous Attendance System using Facial Recognition

    Traditional Attendance system is tedious as it is time consuming and there is a more possibilities of human error. Facial Recognition can be used for attendance in institutions and other organisations which is less time consuming and more effective in crowd. This project includes image processing and artificial intelligence to detect the faces and mark their attendance in a spreadsheet. The project comes with a gui which can be used by anyone easily.

    PROJECT

    Project Rebuild

    Project Rebuild is an initiative to rebuild the society during the worldwide pandemic COVID-19. It includes an autonomous Health Care Robot. In this Robot, my contribution is on SLAM and autonomous navigation using ROS.
